# APJ STS ASQ Triage

## Overview

Skills for triaging incoming and follow-up ASQs (Ask STS requests) for the APJ STS Team. Handles new ASQ intake, validation of use case stages, workspace setup checks, engineer assignment recommendations, and requestor follow-up via SFDC chatter.

## Skills

- **new-asq-triage** — Triage incoming ASQs: validate use cases, check stages, recommend engineer assignments
- **asq-followup-triage** — Follow up on On Hold/Under Review ASQs: check for responses, re-validate stages, send reminders
